🆘 TROUBLESHOOTING

   Issue: npm: command not found
   → Node.js not enabled in cPanel
   → Contact hosting provider to enable Node.js support

   Issue: npm install fails
   → Run: npm install --legacy-peer-deps --force
   → Check disk space: df -h

   Issue: build fails
   → Check logs: npm run build
   → Ensure TypeScript config is correct

   Issue: Port already in use
   → Next.js default port is 3000
   → Configure port: PORT=8080 npm start
